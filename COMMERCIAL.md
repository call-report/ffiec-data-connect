# Commercial Licensing & Support

## You Don't Need a Commercial License to Use This Library

`ffiec-data-connect` is free and open-source software released under the [Mozilla Public License 2.0](LICENSE). The MPL 2.0 already grants you broad rights at no cost, including:

- **Commercial use** — use the library in for-profit products, internal tools, and client work
- **Modification** — fork it, patch it, extend it, adapt it to your needs
- **Distribution** — ship it as part of your own software or redistribute it unchanged
- **Private use** — run it in production, in closed-source environments, inside your organization

You do not need permission, a license key, or a contract to do any of the above. The full text of the license is available in the [LICENSE](LICENSE) file and at <https://mozilla.org/MPL/2.0/>. If MPL 2.0 works for you, you are already done — install it, use it, build things with it.

## Why Offer a Commercial License Then?

Some organizations — particularly banks, regulated financial institutions, and other firms with formal compliance, procurement, or audit requirements — need things that an open-source license simply does not provide. An OSI-approved license like MPL 2.0 is, by design, an "as-is, no warranty" grant. It does not include a phone number to call, a response-time SLA, an indemnification clause, or a contract that a vendor risk management team can sign off on.

The commercial offering exists to fill that gap. It is a **paid support and services agreement** layered on top of the same open-source library that everyone else uses. It is entirely optional, and it does not change what the free library can do.

## The Open-Source Library Is Not Going Anywhere

To be explicit about something that matters to us: **the commercial offering does not reduce or gate the functionality of the free library in any way.**

- The MPL 2.0 version on PyPI and GitHub will remain the primary distribution.
- There is no license check, no telemetry phoning home, no "pro" features hidden behind a paywall, and no plan to introduce any of those things.
- Bug fixes and improvements that come out of commercial engagements flow back into the open-source codebase wherever the customer permits it.
- Community contributions, GitHub issues, and public discussion remain a first-class way to interact with the project.

Civic Forge Solutions LLC is committed to keeping `ffiec-data-connect` genuinely free and open. The commercial license is a way for organizations that need a supported vendor relationship to get one — nothing more.
